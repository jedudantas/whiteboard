Tags estruturais e genéricas para HTML 5

Tag Main

A tag main tem como função agrupar e marcar o conteúdo principal da página que você está lendo nesse momento. Normalmente ele é um item único, cada página tem seu próprio main.

Tags Header e Footer

A tag header é um cabeçalho da página, enquanto o footer é um rodapé. Suas funções estruturais são especialmente úteis para organizar o seu conteúdo.

Tag Section

A tag section gera um pedaço genérico de divisória dentro do conteúdo. Tecnicamente falando tudo o que vimos até agora são sections.

Tag Article

A tag article é uma tag genérica que é autossuficiente. Sua principal função é separar por valor semântico. Uma outra forma de interpretar é como um modelo de seção, como artigos num blog.

Tag Nav

A tag nav cria menus de navegação para dentro e fora do site.

Tag Aside

A tag aside cria um espaço onde podemos usar informação superflua para o conteúdo.

Tags Div e Span

Div e Span são tags estruturais usadas para organizar sua página. Divs podem substituir qualquer seção da sua página, criando um bloco de conteúdo. li/lis são pedaços de conteúdo que você quer separar sem perder o fluxo do texto.

Tags textuais para HTML 5

Tags H1~H6

As tags de h1 a h6 são tags que indicam o título e o subtítulo de um texto. Elas são especialmente importantes quando levamos em consideração a hierarquia da informação que você está usando na sua página.

Tag P

A tag p organiza seu conteúdo em um parágrafo dentro do texto.

Tag Br

A tag br representa uma quebra de linha, sendo uma tag que pode ser autocontida. Ela força uma quebra de linha no texto, fazendo com o que o texto continue na linha seguinte independentemente de onde esteja.

Tag A

A tag "a", também chamada de tag âncora, dá as propriedades de um link para o conteúdo dentro dela, seja texto ou imagem. Você cria o link através do atributo href dentro da tag, onde você pode navegar tanto nos documentos locais, quanto no documento atual ou indo para fora do seu site.

Tags Ul e Ol

Essas tags nos permitem usar listas para organizar nossa informação, seja de forma desordenada ou ordenada. Listas podem conter tanto texto quanto imagem.

Tag Table

Essa tag gera uma tabela, permitindo que sejam feitos registros tabulados. Antigamente se aproveitava da sua organização para construir o layout dos sites. Você pode usar rowspan e colspan para combinar suas células. A tag table possui algumas tags internas para sua organização:

  • Caption - O título da tabela
  • Thead - O cabeçalho da tabela, é preenchido com a tag th
  • Tbody - O corpo da tabela. Onde você pode usar a tag tr para criar uma linha e a tag td para criar uma célula.
  • Tfooter - Funciona como o rodapé da tabela, serve para tabelas que fiquem grandes demais

Tags de Arquivo

Tag Img

Tag para inserir imagens no seu documento. É uma tag autofechada, onde eu posso definir a fonte e seu tamanho. Você pode usar tanto uma imagem local quanto online.

Tag Figure

Tag para dar destaque a um conteúdo independentemente de ser imagem, texto, ou vídeo.

Tag Form

Tag de interação que permite o usuário da página enviar informação para o site. Quando usando essa tag você precisa escolher quais serão os formatos usados no input do usuário através do input type, a forma de transmissão da informação através do method e se essa pesquisa vai ser enviada para outro canto através do action.

CSS

Usos do CSS

O CSS veio para normatizar a estilização dos sites. Dentro dele nós usamos seletores para escolher um pedaço do nosso documento e blocos de instrução para modifica-lo. Nós temos 3 formas de aplicar o css ao HTML:

  • Inline (Tag Style) - Para elementos isolados ou corner cases do documento.
  • Stylesheet Externa - Serve para aplicar o mesmo CSS para diversas páginas HTML.
  • Stylesheet Interna - Normalmente usada quando o CSS de uma página vai ser mais único.
